# Application: Bintulu Smart City

## Overview

AI-powered smart city solution for Bintulu, Malaysia. Two primary use cases: adaptive traffic signal control and smart parking management.

## Platform Features Used

| Feature | Platform Doc | Purpose |
|---------|-------------|---------|
| Traffic Signal Control | [traffic-signal_control](../../../03_platform/traffic-signal_control.md) | Adaptive traffic light timing |
| Smart Parking | [traffic-smart_parking](../../../03_platform/traffic-smart_parking.md) | Parking bay occupancy detection |
| Zone Intrusion | [access-zone_intrusion](../../../03_platform/access-zone_intrusion.md) | No-parking zone enforcement |
