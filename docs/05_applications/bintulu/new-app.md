We are looking at this AI Traffic Light main features as below:
Real‑time vehicle detection and classification using AI such as car, motorcycle, truck, bus etc.
Adaptive signal timing based on traffic density to reduce congestion.
Multi‑lane traffic flow monitoring for optimized phase control (monitor each lane of junction separately).
Incident and anomaly detection (stalled vehicles, blockages). 
Priority routing for emergency or authorised vehicles.
Integration with the central traffic management platform and scalable for city-wide deployment.
Whereas in Smart Parking main features using Edge AI solution on existing CCTV, including key cost‑saving benefits:
Smart Parking – Main Features
AI vehicle detection and bay occupancy monitoring.
Automatic license plate recognition (ANPR) for smoother entry/exit.
AI‑based safety monitoring (loitering, intrusion, accidents).
Detection of parking violations (double parking, obstruction).
Real‑time guidance on available spaces.
Centralized dashboard with analytics for operations and planning.
Cost‑Saving Benefits
Minimal new hardware required by reusing existing CCTV.
Lower manpower and enforcement costs through automation.
Reduced maintenance expenses compared to ground sensors.
Higher revenue accuracy and improved parking utilization.
